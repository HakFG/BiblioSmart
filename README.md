## Sobre o projeto

O BiblioSmart é uma plataforma web que automatiza empréstimos,
reservas, catalogação e gestão de usuários de bibliotecas universitárias.

## Tecnologias previstas

- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express.js
- Banco de dados: PostgreSQL
- Autenticação: JWT + bcrypt

## Estrutura do repositório

- `docs/` — Relatório técnico e diagramas UML
- `prototype/` — Protótipo HTML navegável
